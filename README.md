This is a very simple library aiming to implement a basic version of SSL.

You have to install Rust, and the included package manager, Cargo.

Components made:
  1. HMAC (SHA-256)
  2. AES
  3. Diffie Hellman

How to run:
  1. Download the repo
  2. cd into it
  3. At the terminal, run "cargo run"
